🤖 Robotics Control Center

Welcome to the Robotics Lab Interface. This website shows robotics projects, tasks, and electronic components used in our systems.

💡 Idea of the Project

Create smart robotic systems using sensors, microcontrollers and automation. Example: Smart Home System, Line Follower Robot, Fire Detection Robot.

🚀 Goal

Design intelligent machines that help humans in everyday life using programming and electronics.

🛠 Robotics Projects

Line Follower Robot

Robot that follows a black line using IR sensors.

Smart Home System

Automation system controlling lights and sensors.

Fire Alarm Robot

Robot that detects fire using flame sensors and alerts users.

📋 Tasks

  • Design circuit diagram
  • Write Arduino code
  • Test sensors
  • Build robot structure
  • Debug and improve system

🔩 Components

Arduino UNO

Main microcontroller board used to control robots.

Ultrasonic Sensor

Measures distance to avoid obstacles.

PIR Motion Sensor

Detects movement in smart systems.

Servo Motor

Controls robotic arm movement.

LED + Buzzer

Used for visual and sound signals.
